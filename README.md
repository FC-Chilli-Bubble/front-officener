## ✨ 기능 소개

###  🔐 인증/인가

### 🚚  함께배달
- 마감임박 게시글 조회
- 카테고리별 게시글 조회
- 내가 참여한 배달 목록 조회
- 나의 채팅 목록 조회
- 함께배달 게시글 상세 조회
- 함께배달 참여 및 채팅방 참여
- 함게배달 게시글 수정/삭제 (호스트)
- 함게배달 게시글 작성
<br/>
- 채팅(웹소켓)
- 채팅 참여자 강퇴 (호스트)
- 채팅 나가기 요청
- 채팅 참여자 신고
<br/>
- FCM 푸시 알림
- 사용자 알림 목록 조회

### 🛗 엘리베이터
- 사용자 빌딩 엘리베이터 목록 조회
- 사용자 지정 엘리베이터 설정
- 엘리베이터 현황 조회
